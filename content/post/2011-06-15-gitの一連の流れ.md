gitの一連の流れをすぐ忘れるのでメモっておく。

## 【リポジトリ側】 {.section}

Dropbox/repo/
  
にリポジトリを作成する
  
mkdir test.git
  
cd test.git
  
git &#8211;bare init

## 【実際のソース側】 {.section}

管理したいソースがあるディレクトリ内に移動し、初期化
  
cd workspace/test
  
git init
  
版管理対象外のファイルを読み込む
  
git config core.excludesfile &#8220;$HOME&#8221;/.git-excludes
  
※版管理対象外とする下記パターンを ~/.git-excludes に記入済み。
  
*~
  
.project
  
.DS_Store
  
.classpath
  
gen/
  
bin/
  
git add .　←①
  
git commit でエディタ起動　←②
  
（書式例）
